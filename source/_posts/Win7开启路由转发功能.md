Win7 开启路由转发功能

[](/notename/ "archive 20140427")

原来以为 windows 只有 server 版本才有这个功能

今天发现 win7 也有

打开控制面板 -> 管理工具 -> 服务 ->Routing and Remote Access, 默认不开启, 开启就 ok 了

可以把两个网卡的网络连接起来
